# sk-enterprise (e-bill)

GST-style invoice PDF app (Next.js). Run locally with `npm install` and `npm run dev`.

Deploy on [Vercel](https://vercel.com): set **`BLOB_READ_WRITE_TOKEN`**, then **redeploy**. Default is a **private** Blob store; if yours is **public**, set **`BLOB_STORE_ACCESS=public`**.
